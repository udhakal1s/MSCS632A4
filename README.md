## Project Overview
This project is a small scheduling application that manages employee work schedules for a company that operates 7 days a week. Employees can choose morning, afternoon, or evening shifts. The program assigns shifts while adhering to company rules, such as maximum workdays and minimum staffing per shift.

The project is implemented in Java and Python to demonstrate control structures and logic

### Tools I Used

- Java JDK -19

### How to Run

2. Open the Java version in IntelliJ IDEA.
3. Make sure to set the Project structure correctly (I chose JDK 19, but JDK 17 and above should be completely fine)
